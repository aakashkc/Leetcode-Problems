Your Current Solution
class Solution(object):
    def missingNumber(self, nums):

        for i in range(len(nums)+1):
            if i not in nums:
                return i
Main DSA Problem

This line:

if i not in nums

is expensive.

Why?

Because list searching is:

O(n)

Python must scan the whole list.

Better DSA Approach #1 — Sum Formula

This is the classic optimal beginner solution.

Key Observation

If numbers from:

0 → n

were complete, their sum should be:

2
n(n+1)
	​


Example
nums = [3,0,1]

Length:

n = 3

Expected sum:

0 + 1 + 2 + 3 = 6

Actual sum:

3 + 0 + 1 = 4

Missing:

6 - 4 = 2
Better Code
class Solution(object):

    def missingNumber(self, nums):

        n = len(nums)

        expected = n * (n + 1) // 2
        actual = sum(nums)

        return expected - actual
Complexity

Time:

O(n)

Space:

O(1)

Much better.
