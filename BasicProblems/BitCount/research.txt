Refactored Version of YOUR Logic
num = int(input("enter number: "))
ans = []

for i in range(num + 1):
    n = i
    count = 0

    while n > 0:
        if n % 2 == 1:
            count += 1
        n //= 2

    ans.append(count)

print(ans)
Improvements Made
Better variable usage
n = i

Never destroy loop variables.

No binary string creation

This reduces space usage.

Time Complexity

Total:

O(n log n)
DSA Perspective Solution (Optimal)

This is the important part.

The real DSA insight is:

For any number:

bits(i) = bits(i // 2) + (i % 2)

Why?

When dividing by 2:

rightmost bit disappears
remainder tells whether last bit was 1

Example:

5 = 101
5 // 2 = 10 = 2
5 % 2 = 1

bits(5) = bits(2) + 1
Optimal DP + Bit Manipulation Solution
class Solution(object):
    def countBits(self, n):
        ans = [0] * (n + 1)

        for i in range(1, n + 1):
            ans[i] = ans[i // 2] + (i % 2)

        return ans

class Solution(object):
    def countBits(self, n):
        ans = [0] * (n + 1)

        for i in range(1, n + 1):
            ans[i] = ans[i >> 1] + (i & 1)

        return ans
Bitwise Meaning
Right Shift
i >> 1

Same as:

i // 2

i & 1
1 → odd
